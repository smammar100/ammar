# Conventions

This folder contains current project conventions that are too detailed for
`AGENTS.md` but still useful before editing code or content.

Use these files as the first reference for how the site should be structured,
written, styled, and verified.

| File | Purpose |
| --- | --- |
| `architecture.md` | Astro, layout, content collection, and React usage conventions |
| `content.md` | Public terminology and frontmatter conventions |
| `styling.md` | Visual design, Tailwind, prose, and interaction conventions |
| `assets.md` | Image and embed conventions |
