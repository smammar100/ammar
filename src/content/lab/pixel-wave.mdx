Pixel Wave is the text reveal used on the home page. Each character starts in a pixel typeface, cycles through a few glyphs, then settles into Geist Sans.

The effect is intentionally small: a tactile moment that gives the interface a little craft without turning the page into an animation showcase.

On the home page, the main name and closing tagline only animate once per session. That restraint keeps the effect from feeling heavy when moving back and forth through the site, while preserving the first impression.
