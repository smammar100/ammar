Airbnb's new 3D icons have a trick to them. Put any two side by side and they read as family — same light, same materials, same little world. That's not luck. That's structure.

So I broke the structure down. A locked camera angle. Consistent lighting. Matte materials. A locked color palette. Airbnb's designers built the rules once and then repeated them — nobody reinvents the system per icon.

Then I asked a different question: could I get that same discipline out of an AI image model, without opening Cinema4D, Blender, or Spline? My move was to write the rules down as a JSON file and make ChatGPT render against it. A style engine instead of a lucky prompt.

The bar I set for the system: every render should look like a sibling of the last one. Not identical — siblings. Same family, different subject.
